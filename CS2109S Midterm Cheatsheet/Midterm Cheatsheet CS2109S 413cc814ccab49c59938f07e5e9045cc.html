Informed, Local, and Adversarial Search

A search algorithm takes in a search problem as input and returns a solution/failure.

It is defined by the order of node expansion. Evaluation criteria:

• Time complexity: number of nodes generated or expanded

• Space complexity: maximum number of nodes in memory

• Completeness: does it return a solution if it exists?

• Optimality: does it always find the least-cost solution?

Main Measures: branching factor (b), depth (d), maximum depth (m)

Three important aspects of a course teaching about AI:

  1. Foundational Understanding: The course should provide a solid understanding of various types of AI, including their principles, strengths, and limitations. This foundational knowledge serves as a basis for comprehending more complex concepts and techniques.
  1. Algorithmic Proficiency: Mastery of both uninformed and informed search algorithms is crucial. These algorithms form the backbone of AI problem-solving methodologies. Students should grasp how these algorithms work, when to use them, and their computational implications.
  1. Applied Techniques: In-depth coverage of advanced techniques such as minimax, alpha-beta pruning, and decision trees is essential. These techniques are widely used in AI applications, particularly in areas like game-playing, optimization, and decision-making. Understanding their implementation and real-world applications enhances students' ability to solve complex AI problems effectively.
